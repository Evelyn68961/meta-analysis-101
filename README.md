# Meta-Analysis 101 — A Beginner's Guide

An interactive, beginner-friendly website that introduces the concept of meta-analysis: what it is, why it matters, and how it's done.

## Features

- **Animated Study Combiner** — watch individual study results merge into a single pooled estimate
- **Interactive Forest Plot** — click on squares, lines, and the diamond to learn what each element means
- **Step-by-Step Methodology** — 8 expandable steps with real-world cooking analogies
- **Knowledge Quiz** — 5 questions with instant feedback and explanations
- **Scroll Animations** — smooth fade-in effects as you navigate

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm start

# Build for production
npm run build
```

## Tech Stack

- React 18
- No external UI libraries — pure React with inline styles
- Google Fonts: Source Serif 4 + Outfit

## Content Sources

Content synthesized from leading methodological guides including PRISMA 2020, Cochrane Handbook, and peer-reviewed meta-analysis tutorials.
